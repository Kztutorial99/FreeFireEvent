  CARA INSTALASI
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  1. Ekstrak file ZIP ini — akan muncul folder CONFIGPATCH
  2. Buka File Manager di HP kamu
  3. Masuk ke:
       Internal Storage
         └── Android
               └── data
                     └── com.dts.freefireth
                           └── files
  4. Salin SEMUA file dari folder CONFIGPATCH ke dalam folder files/
  5. Jika diminta timpa file lama, pilih YA / REPLACE ALL
  6. Tutup File Manager
  7. Buka Free Fire dan masuk ke lobby
  8. Konfigurasi aktif otomatis saat match dimulai

  CATATAN:
  - Gunakan File Manager bawaan HP atau MT Manager / Solid Explorer
  - Pastikan Free Fire sudah versi 6.1.3 (OB53)
  - Jika config belum aktif, clear cache Free Fire lalu restart HP

  ⚠  CONFIG TIDAK BEKERJA?
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  Kemungkinan penyebab:
  - Android versi di bawah 6.0
  - RAM kurang dari 2 GB
  - Free Fire belum diupdate ke versi OB53
  - File diletakkan di path yang salah

  Solusi:
  1. Pastikan path: Internal/Android/data/com.dts.freefireth/files/
  2. Update Free Fire ke versi terbaru
  3. Clear cache Free Fire → restart HP → buka game
  4. Nonaktifkan Google Play Protect sementara
  5. Coba install ulang config menggunakan MT Manager
